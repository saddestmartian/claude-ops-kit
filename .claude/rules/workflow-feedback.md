### Complexity profile is user choice, not auto-detected (2026-04-10)
Auto-detect gathers signals and recommends, but NEVER selects the profile automatically. The user must explicitly choose Solo/Team/Enterprise. Confirmed during onboarding skill design.

### Vibecoder context changes profile recommendation (2026-04-10)
A non-developer user benefits MORE from structured rules, not less. The Solo profile assumption (fewer rules = better for speed) only holds for experienced devs. For vibecoders, Team profile provides the instincts they're supplementing with Claude.
